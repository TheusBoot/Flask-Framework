# Flask-Framework
simples API em flask


## Instalando O FLASK
```python
pip install flask
```


## Rodando o FLASK

```python
export FLASK_APP=API.py
flask run
```
## Modo DEBUG ON 

```python
export FLASK_ENV=development
run flask
```
